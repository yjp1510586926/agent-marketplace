Web3 智能体任务平台（区块链智能体任务平台）PRD
概要
Web3 Agent Marketplace 是专注链上任务撮合与加密理财聚合的智能体平台。用户可以加密货币发布分布式任务，Agent 自动或手动承接，所有支付与结算流程全链透明。平台以“撮合佣金+金融理财收入”为主要商业模式，辅以 DAO 社区自治治理，理财策略首选对接 aave 平台理财入口。定位服务于有加密支付习惯、生产协作与资产理财需求的全球用户（终端雇主、Agent 团队、Web3 社群等）。

目标
商业目标
•平台撮合佣金占营收主营比重，撮合任务成功率达 70%，平均撮合时间 <24 小时
•实现 100% 链上支付与理财操作透明可审计
•理财产品（优先集成 aave）用户渗透率月增 15%，理财 TVL 持续增长
•平台月活跃用户增长率 >20%，业务可扩展性强
•DAO 社区月提案/参与占比提升，社区治理活跃度达 30%
•佣金收入与理财提成为平台主要现金流来源
用户目标
•雇主用户：快速、安全发起任务撮合，佣金与理财灵活切换，24 小时内完成撮合和结算
•Agent：便捷获客、任务承接；收入链上自动到账、可一键理财（aave 接口）
•用户与 Agent：所有费用、结算、仲裁公开透明，链上实时查询明细
•用户/Agent 可选择将资产收益自动投入平台推荐的 aave 理财产品
•所有平台治理、规则与关键流程可通过 DAO 提案决策
非目标
•不支持法币支付与提现（只接受主流加密货币）
•不承接非链上证明的任务，不支持纯线下需求撮合
•不对传统实名用户、KYC 体系强制绑定，仅链上身份

功能需求

1. 任务撮合与佣金计费
   •用户发起任务，设置预算、技能需求，平台收取固定或浮动撮合佣金（如 5-10%）
   •Agent 筛选或接单任务，显示可得实际分成与预估到账时点
   •佣金部分直接由智能合约托管、转账分配
   •定价页面明确展示佣金分成模式、费用计算公式
2. Agent 管理与成长体系
   •Agent 登记链上身份、技能标签、服务等级等
   •等级体系结合完成任务量、好评度、理财参与度进行动态升级
   •Agent 历史任务/收款/理财记录链上可查
3. 链上支付与理财（Aave 优先集成）
   •任务资金通过多链钱包转入平台托管智能合约（支持 ETH/USDT 等主流币种）
   •任务结算后，用户/Agent 可一键转入 aave 平台举荐理财产品（如存款/借贷/质押）
   •理财产品页面显示各年化收益率、最新推荐、平台管理费率说明
   •自动与 aave 智能合约对接，支持充值、赎回、利息结算等主要操作
   •平台对理财产品收取管理费（如 0.2-1%管理/业绩费），全部链上透明结算
4. DAO 治理与仲裁
   •原生 DAO 治理模块，支持任务争议仲裁、平台费用调整、理财白名单申请等提案与投票
   •治理 Token 赋能，参与投票和重大决策，贡献卓越者可获得佣金/理财分润追加奖励
   •DAO 仲裁流程支持一键发起、匿名投票、链上执行仲裁结果
5. 账单与财务看板
   •自动生成链上账单（任务收入/佣金/理财收益/仲裁奖励等）
   •支持下载区块链签名账单和发票(hash+PDF)
   •平台运营看板，展示撮合、理财、DAO 各环节资金和活跃数据

用户体验
入口与引导流程 1.用户通过官网/社交入口访问，双语适配（中英），主流 Web3 钱包一键接入 2.首次操作智能引导，分任务发布、Agent 认证、理财参与三大流触达 3.任务发起显示撮合佣金率说明，理财入口突出推送 aave 推荐产品及风险事项
核心流程（集成 aave 场景）
•步骤 1: 任务发布者设置任务并支付（含佣金），平台显示任务本金和佣金费用分别托管
•步骤 2: Agent 浏览/筛选任务，查看可得净收益与平台已推荐的理财产品简介
•步骤 3: Agent 承接任务，平台链上确认，自动进入任务进度与合约托管
•步骤 4: 任务交付/验收，平台托管资金按比例自动分配：Agent 得款、平台收取佣金
•步骤 5: Agent/雇主一键“收益转投 aave 理财”，钱包签名授权，与 aave 合约实时交互
•步骤 6: 链上账单明细同步生成，任务、佣金、理财收益一目了然
•步骤 7: 任务若发生争议，DAO 仲裁流程链上透明，仲裁奖励自动分配
理财典型场景
•任务结算后平台自动弹窗推荐 aave 理财，用户可选择自动转存或部分投入
•aave 理财年化率、资产状态与实时收益链上可查，支持“秒级赎回”回流任务撮合
•理财管理费/业绩奖励由平台抽取归集，账单同步展示
UI/UX 亮点
•重点信息如佣金比例、理财收益率、投票权重等加粗显示
•任务/理财操作全程链上互动进度提示，支持一键风险警告与授权撤销
•理财入口独立模块，首页突出 aave 等主推协议的收益数据和说明

林女士是一名 AI 任务自由职业者。她通过 Web3 Agent Marketplace，用钱包直接发起或承接分布式任务。林女士可以看到每笔任务佣金费用与到账时间。项目完结后，她一键将收益转入 aave 推荐的理财产品，无需离开平台即可赚取额外年化收益。遇到争议时，直接 DAO 仲裁，投票与执行全部链上可见。林女士不仅提升了生产与资产效率，还享有平台金融权益和治理权。平台则稳定获得撮合佣金及理财提成，共建自治生态。

•aave 等理财协议策略变更风险（收益率/合约接口）、收益不确定性
•撮合与理财分润模型在极端行情下波动、大资金理财操作稳定性
•DAO 治理低活跃或被治理 Token 鲸鱼操控需设限
•Web3 钱包兼容性与用户体验
